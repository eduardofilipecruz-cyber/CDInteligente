Fundamentos de Python

Conceitos essenciais para começar a programar em Python.

Variáveis

Explicação
Variáveis são espaços na memória usados para armazenar valores, como números, textos ou listas. Em Python não é necessário declarar o tipo: ele é definido automaticamente.
Exemplo
nome = "Maria"
idade = 21
altura = 1.68
Curiosidade
Python usa tipagem dinâmica: a mesma variável pode receber um número e, depois, um texto, sem gerar erro.

If

Explicação
A estrutura if executa um bloco de código apenas quando uma condição é verdadeira.
Exemplo
idade = 18
if idade >= 18:
    print("Maior de idade")
Curiosidade
Python não usa chaves {} nem parênteses obrigatórios: a indentação define o bloco de código.

Else

Explicação
else define o que acontece quando a condição do if é falsa. Pode ser combinado com elif para múltiplas condições.
Exemplo
nota = 6
if nota >= 7:
    print("Aprovado")
elif nota >= 5:
    print("Recuperação")
else:
    print("Reprovado")
Curiosidade
Python não possui switch/case tradicional; a partir da versão 3.10, existe a alternativa match/case.

For

Explicação
O laço for percorre elementos de uma sequência (lista, string, intervalo) repetindo um bloco de código.
Exemplo
for numero in range(5):
    print(numero)
Curiosidade
A função range() não cria toda a lista em memória de uma vez — ela gera os números sob demanda.

While

Explicação
O laço while repete um bloco enquanto uma condição permanecer verdadeira.
Exemplo
contador = 0
while contador < 3:
    print(contador)
    contador += 1
Curiosidade
Esquecer de atualizar a variável de controle é a causa mais comum de loops infinitos.

Funções

Explicação
Funções agrupam blocos de código reutilizáveis, definidas com def, podendo receber parâmetros e retornar valores.
Exemplo
def somar(a, b):
    return a + b

resultado = somar(2, 3)
Curiosidade
Funções em Python podem retornar múltiplos valores ao mesmo tempo, como em uma tupla: return a, b.

Listas

Explicação
Listas armazenam coleções ordenadas de itens, que podem ser de tipos diferentes e modificadas após a criação.
Exemplo
frutas = ["maçã", "banana", "uva"]
frutas.append("pera")
print(frutas[0])
Curiosidade
Listas em Python podem misturar tipos na mesma estrutura: [1, "texto", 3.5, True] é válido.

Tratamento de erros

Explicação
Blocos try/except capturam erros em tempo de execução, evitando que o programa pare inesperadamente.
Exemplo
try:
    numero = int(input("Digite um número: "))
except ValueError:
    print("Entrada inválida")
Curiosidade
É possível usar finally para executar código independentemente de ocorrer erro ou não.

Modularização

Explicação
Modularizar é dividir o código em arquivos e funções menores e reutilizáveis, usando import para conectá-los.
Exemplo
# arquivo: utilidades.py
def saudacao(nome):
    return f"Olá, {nome}!"

# arquivo: principal.py
from utilidades import saudacao
print(saudacao("Ana"))
Curiosidade
Cada arquivo Python é, por padrão, um módulo independente que pode ser importado em outro.
